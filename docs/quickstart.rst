===============
Getting Started
===============

Installation
=============

Use your favorite Python package manager to install the app from PyPI, e.g.

Example::

    pip install djangobricks


Configuration
=============

Add ``djangobricks`` to the ``INSTALLED_APPS`` within your settings file
(usually ``settings.py``).

Example::

    INSTALLED_APPS = [
        [...]
        'djangobricks',
    ]
